Code Generated by LLM

In the development of this project, an LLM was used to generate the following parts of the application:

Carbon Footprint Calculator

The LLM provided the initial implementation for the Carbon Footprint Calculator, including the form fields and the logic to calculate a user's carbon footprint based on inputted daily activities like driving and electricity use.

Water Usage Calculator

The Water Usage Calculator page was also generated by the LLM. This includes the form that calculates water consumption based on user-provided data, like minutes spent showering and the number of dishes washed, and the logic to display the result.

Products Page

The LLM helped with the implementation of the Products page, with the structure of every individual page.

Learn More Page (Text Content)

The descriptive text on the “Learn More” page was generated by the LLM. It explains the benefits of sustainable living and provides tips and information on how users can reduce their environmental impact.
